Build a modern calendar event management application that integrates with Google Calendar to provide full CRUD operations for events. The application features a dual-view interface (list and calendar views) with automatic Google Meet link generation, timezone support, and participant management. The design follows a Modern SaaS Productivity Pattern (inspired by Linear/Notion) for a clean, efficient user experience.

Technology Stack
Frontend:

React with TypeScript and Vite
Wouter for routing (lightweight React Router alternative)
shadcn/ui (Radix UI primitives)
Tailwind CSS with custom HSL color system
TanStack React Query v5 for server state management
React Hook Form with Zod validation
DM Sans font (Google Fonts) for primary text, Fira Code for monospace
Dark mode support with theme provider
Backend:

Express.js with TypeScript on Node.js
Native Node.js http module for server creation
Google Calendar API (via googleapis library)
Replit Connector for OAuth management
Zod for runtime validation
Build & Deployment:

Vite for frontend bundling and development
esbuild for server-side bundling
PostgreSQL with Drizzle ORM (configured but not actively used for event storage)
Data Models & Schemas
Event Schema (shared/schema.ts)
export const eventSchema = z.object({
  id: z.string(),
  title: z.string().min(1).max(100),
  startTime: z.string(), // ISO 8601 format
  endTime: z.string(), // ISO 8601 format
  timezone: z.string(), // IANA timezone name
  participants: z.array(z.string().email()).max(6),
  meetLink: z.string().url().optional().nullable(),
  recordingUrl: z.string().url().optional().nullable(),
  description: z.string().optional().nullable(),
});
export const insertEventSchema = z.object({
  title: z.string().min(1, "Title is required").max(100, "Title must be 100 characters or less"),
  startTime: z.string().min(1, "Start time is required"),
  endTime: z.string().min(1, "End time is required"),
  timezone: z.string().min(1, "Timezone is required"),
  participants: z.array(z.string().email("Invalid email address")).max(6, "Maximum 6 participants allowed"),
  description: z.string().optional().nullable(),
}).refine((data) => {
  const start = new Date(data.startTime);
  const end = new Date(data.endTime);
  return end > start;
}, {
  message: "End time must be after start time",
  path: ["endTime"],
});
export const organizerSchema = z.object({
  email: z.string().email(),
  name: z.string().optional(),
});

API Endpoints
GET /api/organizer
Returns authenticated user's Google account information.

Response: { email: string, name?: string }
GET /api/events
Lists all calendar events from 30 days past to 365 days in the future.

Response: Array of Event objects
POST /api/events
Creates a new event with automatic Google Meet link generation.

Request Body: { title, startTime, endTime, timezone, participants, description? }
Response: Created Event object
Features: Sends invitations to participants, creates Google Meet link
PUT /api/events/:id
Updates an existing event while preserving the Google Meet link.

Request Body: Same as POST
Response: Updated Event object
DELETE /api/events/:id
Deletes an event from Google Calendar.

Response: 204 No Content
